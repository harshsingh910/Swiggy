 Swiggy Clone - Food Delivery App
Description
This is a Swiggy clone built with React, Redux, and React Router that replicates the core functionality of India's popular food delivery platform. The application allows users to:

Browse restaurants and their menus

Search for specific dishes

Add items to cart with quantity adjustment

View and manage cart items before checkout

Filter menu items by Veg/Non-Veg options

The app features a responsive design with shimmer loading effects, error handling, and a Redux-powered cart management system.

Key Features
Restaurant Listing: Displays nearby restaurants with ratings, delivery time, and cuisine types

Menu Display: Shows categorized restaurant menus with item details and pricing

Cart Functionality: Add/remove items, adjust quantities, and view order summary

Search: Find specific dishes across multiple restaurants

Filters: Veg/Non-Veg filters for menu items

Responsive UI: Works on various screen sizes

Loading States: Shimmer effects while data loads

Technologies Used
Frontend: React.js

State Management: Redux Toolkit

Routing: React Router

Styling: Tailwind CSS

API Integration: Swiggy's public APIs (with CORS proxy)
