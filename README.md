Swiggy Clone - Food Delivery App

Description
1.This is a Swiggy clone built with React, Redux, and React Router that replicates the core functionality of India's popular food delivery platform. The application allows users to:
2.Browse restaurants and their menus
3.Search for specific dishes
4.Add items to cart with quantity adjustment
5.View and manage cart items before checkout
6.Filter menu items by Veg/Non-Veg options

The app features a responsive design with shimmer loading effects, error handling, and a Redux-powered cart management system.

Key Features
1.Restaurant Listing: Displays nearby restaurants with ratings, delivery time, and cuisine types
2.Menu Display: Shows categorized restaurant menus with item details and pricing
3.Cart Functionality: Add/remove items, adjust quantities, and view order summary
5.Filters: Veg/Non-Veg filters for menu items
6.Responsive UI: Works on various screen sizes
7.Loading States: Shimmer effects while data loads

Technologies Used
1.Frontend: React.js
2.State Management: Redux Toolkit
3.Routing: React Router
4.Styling: Tailwind CSS
5.API Integration: Swiggy's public APIs (with CORS proxy)
